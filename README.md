# FCQ-Delivery(Курьерская служба)
Роли: Администратор, Менеджер, Клиент.
Администратор управляет пользователями:
 - создает новых пользователей(admin, manager, customer),
 - изменяет статус(блокирует/активирует)текущих пользователей.
Менеджер управляет курьерами, обрабатывает заявки от клиентов:
 - создает новых курьеров;
 - редактирует данные уже созданных курьеров;
 - просматривает заявки на доставку грузов;
 - назначает курьера на заявку, изменяет статус заявки и подтверждает заказ;
 - просматривает список всех курьеров.
Клиент делает заказы на оставку товара:
 - просматривает информцию о имеющихся автомобилях и тарифах;
 - создает заявку на доставку товара;
 - изменяет статус заявки, когда заказ доставлен.
