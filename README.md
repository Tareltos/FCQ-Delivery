# FCQ-Delivery(Курьерская служба)
Роли: Администратор, Менеджер, Клиент.
Администратор управляет пользователями:
 - создает новых пользователей(admin, manager, customer),
 - изменяет статус(блокирует/активирует)текущих пользователей.
Менеджер управляет курьерами, обрабатывает заявки от клиентов:
 - создает новых курьеров;
 - редактирует данные уже созданных курьеров;
 - просматривает заявки на доставку грузов;
 - может усеществлять фильт по статусу заявки и клиенту;
 - назначает курьера на заявку, вводит расстояние;
 - просматривает список всех курьеров.
Клиент делает заказы на оставку товара:
 - просматривает информцию о имеющихся автомобилях и тарифах;
 - создает заявку на доставку товара;
 - до назанечения менеджером курьера может удальть заявку безвозвратно, после назначения курьера, может только отменить;
 - после назначения менеджером курьера на заявку получает уведомление на почту о необходимости оплатить подтвердить и оплатить заказ;
 - в описании заявки подтверждает и оплачивает доставку;
 - после оплаты, на почту водителя отправляется письмо с данными о деталях доставки;
 - после подтверждения менеджером и до оплаты может отменить заявку, вводит причину отмены, назначается статус "отменена";
 - изменяет статус заявки, когда заказ доставлен.
